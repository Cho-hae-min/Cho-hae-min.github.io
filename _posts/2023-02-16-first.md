- **static** : 난 선언되는 순간, 메모리 할당 이미 되어 있어서 따로 객체 안 만들어도 됨. 
- **일반 ~** : 난 메모리 할당 따로 해줘야 함. 그래서 객체 따로 만들어야 함. (ex. main 함수에 따로, **클래스명 변수 = new 클래스명** 이렇게 해줘야 함.)
